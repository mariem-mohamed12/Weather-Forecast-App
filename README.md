# Weather Forecast App

A modern weather web application that shows current weather and a 3-day forecast.

The app detects the user’s current location using Geolocation and allows searching for any city to display weather details.

## Features

- Get current location weather automatically
- Search weather by city name
- Display today’s weather details
- 3-day weather forecast (today, tomorrow, day after tomorrow)
- Show temperature, condition, humidity, wind speed, and wind direction
- Responsive and clean UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Font Awesome
- Google Fonts
- WeatherAPI
